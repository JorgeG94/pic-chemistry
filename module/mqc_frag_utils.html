Provides combinatorial functions and algorithms for generating molecular fragments, managing fragment lists, and performing many-body expansion calculations. Binomial coefficient calculation Generate sequential monomer indices Generate all fragments up to max level Calculate total number of fragments Generate next combination in sequence Hash-based lookup table for fast fragment index retrieval
