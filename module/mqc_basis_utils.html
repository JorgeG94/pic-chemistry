Provides utilities for normalizing basis set names and locating basis set files

Normalization rules: * -> s (e.g., 6-31G* -> 6-31Gs) + -> p (e.g., 6-31+G -> 6-31pG) (d,p) -> dp (remove parentheses and commas)
