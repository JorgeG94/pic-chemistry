public function normalize_basis_name(basis_name) result(normalized)

Normalize basis set name to filename-safe format Rules: * -> s, + -> p, remove parentheses and their contents Examples: 6-31G -> 6-31Gs 6-31+G -> 6-31pGs 6-31G(d) -> 6-31Gd 6-311G(d,p) -> 6-31Gdp 6-311++G** -> 6-311ppGss
